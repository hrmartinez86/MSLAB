Sample: dhtmlxGrid Tab Order and Readonly Cells dhtmlxGrid main page

enableEditTabOnly method allows to navigate only between editable cells.
And by setExternalTabOrder method you can set external elements, which get focus when tab|tab+shift is pressed in the last|first cell.

enable | disable EditTabOnly 
 
<input type="text" id="elem1" value="start" size="20">
<div id="gridbox" width="400px" height="250px" style="background-color:white;overflow:hidden;"></div>
<input type="text" id="elem2" value="end" size="20">
 
 
<script>
    mygrid = new dhtmlXGridObject('gridbox');
    mygrid.setHeader("Column A, Column B, column c");
    mygrid.setColTypes("ro,ed,ed");
 
    mygrid.enableEditTabOnly(1);
 
    mygrid.setExternalTabOrder("elem1","elem2")
    ....
<script>
