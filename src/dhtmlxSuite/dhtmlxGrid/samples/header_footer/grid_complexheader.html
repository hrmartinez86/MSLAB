Setting multirow header with script
Extend Header
 
    //add second row to header
    mygrid.attachHeader("#rspan,Title,Author,#rspan,#rspan,#rspan,Bestseller,Published");
    mygrid.setSizes();
    //#rspan - include cell in rowspan
    //#cspan - include cell in colspan
