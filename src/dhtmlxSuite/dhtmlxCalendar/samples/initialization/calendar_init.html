Sample: dhtmlxCalendar Initialize object on page dhtmlxCalendar main page

You can place this JavaScript calendar anywhere on your web page, attaching it to any object.
Set up calendar parameters to determine the look of the calendar. 

 
    mCal = new dhtmlxCalendarObject('dhtmlxCalendar');
    mCal.setYearsRange(2000, 2500);
    mCal.draw();
    
    //with editable year and month
    mCal2 = new dhtmlxCalendarObject('dhtmlxCalendar2', false, {isYearEditable: true,isMonthEditable: true});
    mCal2.setYearsRange(2000, 2500);
    mCal2.draw();
