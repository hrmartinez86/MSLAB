Sample: dhtmlxTabbar Loading in iframes dhtmlxTabbar main page

Tabbar content can be loaded in iframes. Set hrefmode attribute value to iframes and add href attributes to tab tags to load tab content in iframe, which will be autoimaticaly created for each tab.

Recreate second tab

 
<?xml version="1.0"?>
<tabbar  hrefmode="iframes">
    <row>
        <tab id="b1" width='100px' href="http://ya.ru">Yandex</tab>
        <tab id="b2" width='100px' selected="1" href="http://google.com">Google</tab>
    </row>
</tabbar>
