Sample: dhtmlxGrid Loading from JS array dhtmlxGrid main page

You can load grid data from JS array.


Reload from JS array object 
 
<div id="gridbox" width="100%" height="250px" style="background-color:white;overflow:hidden"></div>
<script>
 
var data=[
    [
        "100",
        "A Time to Kill",
        "John Grisham"
    ],
    [
        "1000",
        "Blood and Smoke",
        "Stephen King"
    ]
        ...
]
    mygrid = new dhtmlXGridObject('gridbox');
    /*loading from JS array*/
    mygrid.load("_data.js","jsarray");
    /*loading from JS array object*/
    mygrid.parse(data,"jsarray");
</script>
