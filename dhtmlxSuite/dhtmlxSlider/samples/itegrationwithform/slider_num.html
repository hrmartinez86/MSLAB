Sample: dhtmlxSlider Controlling values dhtmlxSlider main page
DHTML JavaScript Slider samples
	There are 3 values which define output value of slider
		a) Limit - maximum value - 100
		b) Step - size of shifting - 1 by default
		c) Shift - this value added to current slider position, so you can define sliders which starts not from 0



 
<input id="input1" type="text">
<script>
 
    //slider 0-100, step 1
    var slider1=new dhtmlxSlider(null, 500);
    slider1.linkTo('input1');
    slider1.init();
</script>


 
<input id="input2" type="text">
<script>
 
    //slider 100-300, step 1
    var slider2=new dhtmlxSlider(null, 500,null,null,100,300);
    slider2.linkTo('input2');
    slider2.init();
</script>
