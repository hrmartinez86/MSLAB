Common issue with XML

If you XML not loaded correctly ( hilarious "incorrect XML" error ) - please be sure that
  • you are using correct url
  • structure of XML is really valid - just point browser to the XML file
  • there is no whitespaces before <?xml declaration
  • if XML generated by server side script it send with content-type = text/xml
If it still not work , then you can start terrorize support team :)
