Sample: dhtmlxToolbar Easy skinable dhtmlxToolbar main page

You can set new skin using setToolbarCSS method

 

 
<style>
    <!-- css class for toolbar container -->
        .toolbar1{ ...}
    
    <!-- css class for toolbar title -->
        .toolbarText1{ ...}
</style>
 
<script>
 
    aToolBar1=new dhtmlXToolbarObject(document.getElementById('toolbar_zone1'),'100%',16,"Demo toolbar");
    aToolBar1.setOnClickHandler(onButtonClick);
    aToolBar1.loadXML("_toolbar1.xml")
    aToolBar1.setToolbarCSS("toolbar1","toolbarText1");
    aToolBar1.showBar();
</script>
