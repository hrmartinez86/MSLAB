dhtmlxColorpicker is an JavaScript color selection control which allows you to choice any color in user-friendly way, offers sophisticated feature set, cross-browser compatibility and script API.

Being 100% client-side solution, dhtmlxColorpicker can be integrated with any server side technology

Main features:

  1. Easy configurable
  2. Existing input controls integration

Supported Browsers:

  • IE 5.x and above
  • Mozilla 1.4 and above
  • FireFox 1.0 and above
  • Opera 9.0 and above
  • Safari 2.0 and above (1.3 with some limitations)
