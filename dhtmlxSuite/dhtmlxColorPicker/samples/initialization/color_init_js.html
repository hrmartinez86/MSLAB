Sample: dhtmlxColorPicker Initialization with Script dhtmlxColorPicker main page

Using script initialization you can initalize colorpicker based on DIV element and specify its parameters directly in script

 
<div id="colorPicker" style="position:absolute;top:150;left:200;"></div>
<script>
 
    //color picker can be initialized in specified container
    var myCP1 = new dhtmlXColorPicker("colorPicker");
            myCP1.setImagePath("../../codebase/imgs/");
            myCP1.init();
</script>
 
<div id="colorPicker2" style="position:absolute;top:150;left:200;"></div>
<script>
 
    var myCP2 = new dhtmlXColorPicker("colorPicker2",true);
            myCP2.setImagePath("../../codebase/imgs/");
            myCP2.init();
</script>
 
<script>
 
    <div id="colorPicker" style="position:absolute;top:150;left:200;"></div>
    //color picker  can be initialized  in custom colors mode
    var myCP = new dhtmlXColorPicker("colorPicker",null,true);
        myCP.setImagePath("../../codebase/imgs/");
        myCP.init()
</script>
